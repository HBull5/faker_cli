# faker_cli

## Instructions

-   Ensure Node.js is installed `https://nodejs.org/en/download/`

    -   You can check if Node is installed with `node --version`

-   Clone repo into root directory `C:\\ || ~/`

-   Modify .bashrc `nano ~/.bashrc`

    -   At end of file add ``alias faker='~/faker_cli/faker.sh'`

-   Modify .profile `nano ~/.profile`

    -   At end of file add `. ~/.bashrc`

-   Restart your terminal
